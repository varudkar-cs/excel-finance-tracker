# excel-finance-tracker
Personal finance tracker built using Excel with data analysis and visualization

## Overview
This project is an Excel-based personal finance tracker to monitor expenses and analyze spending patterns.

## Features
- Expense tracking
- Total spending calculation (SUM)
- Spending status (IF)
- Category-wise analysis (SUMIF)
- Visualization with pie chart

## Skills Used
- Microsoft Excel
- Data Analysis
- Data Organization

## Purpose
Built as a beginner project to apply Excel concepts practically.
